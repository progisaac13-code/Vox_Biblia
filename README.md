# 🙏 Devocional App

Aplicação web desenvolvida para ajudar pessoas a terem um momento diário com Deus, através da leitura bíblica e anotações pessoais.

---

## 🌅 Sobre o Projeto

O **VOX BÍBLIA** permite que o usuário:

- Leia versículos ou trechos da Bíblia 📖  
- Escreva suas reflexões ✍️  
- Organize suas anotações 🎨  
- Compartilhe conteúdos 📤  

Tudo isso em uma interface simples, limpa e focada na experiência espiritual.

---

## ✨ Funcionalidades

- 📖 Versículo aleatório ou por capítulo  
- 📚 Leitura de trechos completos (não só um versículo)  
- ✍️ Criação e edição de anotações  
- 🎨 Personalização com cores  
- 🔍 Filtro e busca de anotações  
- 📤 Compartilhamento via WhatsApp  
- 🖼️ Exportação de anotação como imagem  
- 🙏 Modo devocional (experiência focada)

---

## 🛠️ Tecnologias utilizadas

- **Backend:** Python + Flask  
- **Banco de dados:** MySQL  
- **ORM:** SQLAlchemy  
- **Frontend:** HTML, CSS, JavaScript
